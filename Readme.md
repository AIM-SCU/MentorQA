# Beyond Factual QA: Mentorship-Oriented Question Answering from Long-Form Multilingual Content

This repository contains the dataset and code for our paper: Beyond Factual QA: Mentorship-Oriented Question Answering from Long-Form Multilingual Content

<p align="center">
  <img src="mentorship.jpg" alt="Mentorhsip Overview" width="50%">
</p>

We introduce MentorQA, the first multilingual benchmark for mentorship-focused question answering from long-form videos, showing that multi-agent QA systems produce more effective guidance than factual QA baselines—especially for complex topics and low-resource languages—while highlighting limits of current automated evaluation.
